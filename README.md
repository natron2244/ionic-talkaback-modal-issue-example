Issue:
Modal content is inaccessible when TalkBack is enabled.

Project: 
This is a starter app from the [Ionic Angular Quickstart](https://ionicframework.com/docs/angular/quickstart) with a modal example from the [Ionic Modal Example](https://ionicframework.com/docs/api/modal).

Steps to Reproduce:
1. Run the project on an Android device
2. Enable TalkBack
3. Open the modal
4. Attempt to interact with any content within the modal

Example Video:
https://github.com/user-attachments/assets/ed2a2a71-49e3-413e-98b6-caedcd5e3e2b

Device Info:
- Device: Pixel 9
- OS: Android 16
- Android Accessibility Suite (TalkBack): 17.0.1.911806202
